The Point Collector!

Game Instructions

You will be given a randm number at the start of the game.


There are four characters below. By clicking on a character you will add a random amount of points to your score.


You win the game by matching you total score to the random number, you lose the game if your total score goes above the random number.


The value of each character is hidden from you until you click on it.


Each time the game starts, the game will change the values of each character.
